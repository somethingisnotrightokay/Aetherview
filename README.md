
> *AetherView* is a free, self-hosted, cross-platform 3D wallpaper engine that turns your screen into a dynamic window into another world. Using webcam face tracking, gyroscope input, and touch drag, it delivers smooth off-axis parallax for desktop and mobile.

Features include:

Panoramic & standard image support with pseudo-3D depth layers

Video backgrounds for dynamic, immersive scenes

Touch drag fallback for mobile / file:// usage

Nebulae, starfields, and floating objects for cinematic space vibes

Lightweight, modular design (works directly in browser via GitHub Pages, no build step required)


Perfect for anyone who wants a magical 3D “window” on their desktop or phone, without paying for a proprietary app.
